# Code Review Standard

All code reviews must perform:

- Runtime Error Audit
- Architecture Audit
- Dependency Audit
- Performance Audit
- Framework Integrity Audit
- Workflow Validation
- Google Apps Script Best Practices Review

Reviewers must:

- Identify errors
- Identify orphaned code
- Identify duplicate logic
- Identify performance bottlenecks
- Identify governance violations

Do not rewrite code during review.
Produce findings and recommendations only.
